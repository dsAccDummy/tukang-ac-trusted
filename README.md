# Tukang AC Trusted

Website direktori teknisi AC terpercaya di Indonesia. Data diambil langsung dari Google Spreadsheet komunitas.

## Tech Stack
- SvelteKit
- Tailwind CSS v4
- svelte-motion (animations)
- shadcn-svelte (UI components)
- PapaParse (CSV parsing)

## Features
- Real-time data dari Google Sheets (auto-refresh 5 menit)
- Search & filter by wilayah, peran, kata kunci
- Smart contact links (WhatsApp, Instagram, telepon)
- Responsive design (mobile, tablet, desktop)
- Smooth animations (entrance, hover, scroll reveal)

## Getting Started
```bash
npm install
npm run dev
```

## Data Source
Data teknisi diambil dari spreadsheet komunitas Tukang AC Trusted.
